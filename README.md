# RTSIM тестовое задание

Форма имеет такие функции, как:

- Поля - email, пароль, подтверждение пароля
- Валидация полей

- Обработка и отображение ошибок
- Адаптивная верстка
- Обязательное использование React и TypeScript
- Использование библиотеки компонентов по желанию
- Работа с mock API (fetch/axios)
